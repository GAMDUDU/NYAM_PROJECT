## 프로젝트 소개

파이널 프로젝트 초기 단계 **4명의 팀원**이 꾸려지게 되었습니다.

저희는 맛집을 공유하는 맛집 추천 사이트를 창작하여 제작하기로 하였습니다.



맛집 리뷰 홈페이지 **NYAM NYAM** 은

맛집을 추천하는 **사용자**와 자신의 가게를 홍보하는 **스토어** 그리고 **관리자**로 나누어져 있습니다.

저는 그중 **관리자**를 담당하였습니다.
